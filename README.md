# Sampling Strategies Framework

Гибкий фреймворк для стратегий семплирования данных в задачах машинного обучения и глубокого обучения.

## Особенности

- **Универсальность**: Работа с временными рядами, табличными данными и гибридными наборами
- **Гибкость**: Легко расширяемая архитектура с поддержкой пользовательских стратегий
- **Производительность**: Оптимизированные реализации для больших наборов данных
- **Интеграция**: Простой API для интеграции в существующие пайплайны
